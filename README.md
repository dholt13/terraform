# Terraform Labs

- Welcome to my portfolio of practical labs completed during the Certified GRC Engineer (CGRCE) course through the GRC Engineering Club. This repository showcases hands-on engineering solutions designed to bridge the gap between traditional security frameworks and modern, automated cloud infrastructure.

- Instead of treating Governance, Risk, and Compliance (GRC) as a checklist, these projects demonstrate how to implement Compliance-as-Code, automate continuous monitoring, and enforce rigid technical controls directly within cloud environments.

# Core Competencies Demonstrated

- Compliance-as-Code (CaC): Automating guardrails using Infrastructure as Code (IaC) to prevent misconfigurations before deployment.

- Control Mapping & Implementation: Translating high-level framework requirements (NIST SP 800-53, ISO 27001, CMMC) into concrete cloud configurations.

- Continuous Auditing & Monitoring: Generating auditable artifacts, cryptographic state verifications, and structured compliance data.

- Secure Cloud Architecture: Provisioning highly secure, isolated, and encrypted storage and identity structures.
